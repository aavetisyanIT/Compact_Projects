# Sticky Navbar

Sticky navs are navigation components that stick to the top of the page as you scroll down.
Here is one with logo popping from the side and page scaling when navbar stuck.
All done with with JS dynamically adding CSS styles.

## Built With

HTML5, CSS3, JavaScript
